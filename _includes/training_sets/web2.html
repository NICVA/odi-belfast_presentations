Evolution of the Web

  • Web 1.0
    • HTTP 1.0 and HTML 1.0-4.0
    • Static documents
    • Curated for invididual consumption
  • Web 2.0
    • HTTP 1.0/1 and HTML 4.0/5.0
    • Content focussed
    • Dynamic to the needs of communities
  • Web 3.0
    • HTTP, SPDY? & HTML 5.0
    • Semantic Web?
    • Ubiquitous?

Standards don't define the webs evolution, people do.

Some experts refer to the web as the worlds biggest social machine.

"Web 2.0 is, of course, a piece of jargon"

- Berners-Lee et. al (2006)

Features of Web 2.0

If we define Web 2.0 as "beyond documents" then there are some key charecteristics:


  • Social networks
  • Content markup
  • Loosely coupled services via APIs
  • Crowed sourced and curated content


Berners-Lee held this as his vision of Web 1.0, hense why Web 2.0 is, to him, jargon. There was nothing new, it just took a while for people to understand the power of the web.
