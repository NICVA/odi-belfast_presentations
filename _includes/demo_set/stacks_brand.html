Using Stacks

Reveal.js allows you to navigate both forwards (you got here) and up and down. Try it by pressing the down arrow!

ODI Branding

In the top left you will notice the ODI brand logo, there are three options which you can apply in the class attribute of your slides:

  • odi-circles (up a slide)
  • odi-polygons (this slide)
  • odi-dynamic (down a slide)

Alignment

Note that you can only use these logos when the text is left aligned on your slide with a class="left" attribute.


You should also use class="odi-logo" on your paragraph (<p>) element to make sure the text doesn't overlap with the logo.
