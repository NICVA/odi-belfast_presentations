Machine readability

  • Digital
  • Structured data
  • Appropriate format

Digital

  • Not all digital data is machine-readable
  • But all machine-readable data is digital
  • PDF: digital, human-readable, but not machine-readable

Structured data

  • All data, even handwritten, has an inherent structure
  • 'Structured data' has a logical structure
  • CSV: represents structural relationship between elements

Appropriate format

  • The format should suit the content
  • Presume users can't read proprietary formats
  • Allows for a wider reuse of the data
  • HTML instead of Word or PDF
  • CSV instead of Excel.
